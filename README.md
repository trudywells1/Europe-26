## Features
- 📱 iPhone PWA — add to home screen for app-like experience
- ⏱ Live countdown to departure
- 🧭 Travel status dashboard
- 📋 Complete booking references with copy buttons
- 📍 Google Maps links for every hotel, restaurant and activity
- 🗂 Collapsible sections for easy navigation
- 🖨 Print-optimised layout

## Setup on GitHub Pages
1. Upload all files to your GitHub repository
2. Go to Settings → Pages → Source: Deploy from branch → main
3. Your site will be live at `https://[username].github.io/[repo-name]/`

## Adding to iPhone Home Screen
1. Open the site in Safari on your iPhone
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add" — it will appear as an app icon

## Files
- `index.html` — main itinerary page
- `app.js` — interactivity, countdown timer, PWA features
- `sw.js` — service worker for offline support
- `manifest.json` — PWA manifest for iPhone home screen
- `images/` — app icons
